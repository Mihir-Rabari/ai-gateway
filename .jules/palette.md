## 2024-04-13 - [Focus Visible Standard]
**Learning:** The console application uses a consistent focus-visible indicator on interactive elements (like icon buttons and utility buttons) to enhance keyboard accessibility.
**Action:** Always apply `focus-visible:outline-none focus-visible:ring-2 focus-visible:ring-white/50` to custom interactive elements in the console to maintain keyboard accessibility standards without compromising the dark monochrome aesthetic.
